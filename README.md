# Software-FJ
Software FJ Management System es una aplicación de escritorio desarrollada en Python para gestionar clientes, servicios y reservas. Implementa Programación Orientada a Objetos, manejo de excepciones, validaciones y registro de eventos, sin utilizar bases de datos, cumpliendo los requisitos de la Fase 4 del curso de Programación de la UNAD.

# Tecnologías utilizadas
Python 3
Tkinter
Programación Orientada a Objetos (POO)
Git
GitHub
